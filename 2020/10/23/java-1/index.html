Dev Club Java Class Lesson 1

欢迎欢迎！这是Java课堂的第一节课。我是XGN，我们来做好学习的准备。

内容：

  • 安装并下载Eclipse
  • 创建Java工程
  • 为以后学习打好基础

课堂准备

请大家先到Here上下载最新版Eclipse Installer。在Installer中下载并安装Java Developer的版本。或者，你可以使用其他IDE，但是本课程不支持。

然后登录Hackerrank并注册该比赛。这里会是以后的（一部分）作业区。

注意：

  • 上课版本为Java8，但是最新的Eclipse支持Java11以上。
  • 你不需要认识Ookami等人是谁
  • 请在QQ上提问
  • 请认真完成作业

创建工程

  1. 打开你的搞基IDE Eclipse，选择Workspace（保存工程文件）的位置。
  2. 在Welcome Page上选择“Create New Java Project”或者在Project View右键New选择Java Project
  3. 填写Project Name，其余选项跳过并选择Finish。如果出现创建module的提示，选择稍后创建。
  4. 完成
