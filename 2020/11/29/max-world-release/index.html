Dev►MC Mod

Max World Release

Mod Info

  • For: Minecraft 1.12.2
  • Library: MinecraftForge
  • Open-source: No

Detail

This simple mod allows you to:

  • multiply the global experience and item drops from mobs

  • multiply mining speed, block drops, global damage multiplier

  • even multiplier of multipliers!

 

Some creative ideas:

  • Make “Minecraft but items are multiplying”

  • Make “Minecraft but all damage are multiplied”

  • Make “Minecraft but each time I get 64 items instead of 1”

 

How to use:

  1. Install the mod.

  2. Open config file and change the parameters.

  3. Run /rbdr and /rmdr in your world
