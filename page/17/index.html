高斯(dick)积分

∫(-inf,inf)e^(-x^2)dx ← 这事高斯积分

原式^2
=∫(-inf,inf)e^(-x^2)dx*∫(-inf,inf)e^(-y^2)dy

=∬e^(-x^2-y^2)dσ D={(x,y)|x∈R,y∈R}

= ∫dθ∫re^(-r^2*sin^2θ-r^2*cos^2)dr D={(r,θ)|r∈[0,inf),θ∈[0,2π]}

=∫dθ∫re^(-r^2)dr

=2π/2

=π

*为甚么这里会多一个r?? → ∫dθ∫re^(-r^2*sin^2θ-r^2*cos^2)dr

搭建博客小记

很早之前就有过这个想法了，但一直没付诸行动。

后来看见梨园论坛上有人出租阿里云美国轻量，配置价格线路还行，就租下来了，闲着没用，就搭起了博客。

很早知道了wordpress，然后宝塔面板又有一个一键部署wordpress源码的插件，就装下来了。整个过程比较顺畅。

博客搭好之后，想着来个https证书吧，然后又偶然发现宝塔有自动申请Let's Encrypt 证书的功能，又加了个Really Simple SSL开启全站SSL。

但https证书一装，整个CSS就全乱了。

如图（网图），按F12看了下，因为许多CSS都没有成功加载，都链到了http://上了，查了一下谷歌在配置文件加一行即可 

if ({% mathjax %}_SERVER['HTTP_X_FORWARDED_PROTO'] == 'https') {% endmathjax %}_SERVER['HTTPS']='on';

加完CSS加载都正常了，可刷新一下显示“网页重定向次数过多”，又查了一下大谷歌，在配置文件添加：

{% mathjax %}_SERVER['HTTPS'] = 'on';define('FORCE_SSL_LOGIN', true);define('FORCE_SSL_ADMIN', true);

即搭建完成，暂时没出现其他问题。（毒奶hhhh）
