新龙人语教程4 - 日期与时间

本文是新外龙人语教程第四集，主要介绍了日期和时间的读法

日期

龙人语可以采用年号纪年法和公元纪年法。

公元纪年法按照中文语序展开即可。例子：2022年12月20日读作eru'rin'eru'eru'nen'shi'eru'e'eru'shi'ri

可以加上人历(re'ri)二字强调：人历2022年12月20日

公元前的说法：(人历)前2022年12月20日

词汇：年月日(nen'e'ri)

年号纪年法先说年号后加时间。例子：自贡元年1月1日读作ji'gon'i'nen'i'e'i'ri

词汇：元(i)

时间

龙人语一开始只有某种crappy的天干地支计时法。但是你也可以用常见的24/12计时法。

24小时计时法：15时10分59秒读作shi'u'shi'shi'fen'u'shi'kyu'myo

词汇：时分秒(shi'fen'myo)

12小时计时法：上午六点朝6时读作zo'ro'shi，下午六点夕6时读作shi'ro'shi

词汇：朝夕(zo'shi)

天干地支法：
| 24小时时间 | 龙人语汉字 | 读法 |
|————|————|——|
| 0-1 | 子 | ji |
| 2-3 | 丑 | cyu |
| 4-5 | 寅 | in |
| 6-7 | 卯 | mo |
| 8-9 | 辰 | chen |
| 10-11| 巳 | shi |
| 12-13 | 午 | mu |
| 14-15 | 未 | me |
| 16-17 | 申 | sen |
|18-19 | 酉 | yu |
| 20-21 | 戌 | syu |
| 22-23 | 亥 | hai |

每两小时中的分钟 龙人语汉字 读法
0-11 甲 kya
12-23 乙 i
24-35 丙 bin
36-47 丁 din
48-59 戊 u
60-71 己 ji
72-83 庚 gen
84-95 辛 shin
96-107 壬 rin
108-119 癸 gi

如果想要再分，可以用刻(ke)，1刻=1分钟，其中每十二分钟最后一刻叫做末刻(mo'ke)，第零刻用……之时(no'shi)表达

例如：

  • 09:13是辰庚一刻(chen'gen'i'ke)
  • 22:18是亥乙六刻(hai'i'ro'ke)
  • 21:12是戌庚之时(syu'gen'no'shi)（可以但不推荐：戌癸零刻(syu'gi'rin'ke)）
  • 21:59是戌癸末刻(syu'gi'mo'ke)（可以但不推荐：戌癸十一刻(syu'gi'shi'i'ke)）

随堂练习！！

翻译下列句子：

  • 下一站是兽一市站。列车到达兽一市站的时间是早上7：40分。
  • 公元2022年8月11日，本教程完工。
  • 每季末日之午甲之时正时、赛所上之丛以查请。
  • 冬之朝6时至夕4时时、戏者者车公站与农所之间以行之时时钘。

可能用到的词汇：

  • 下一站：下一站(sya'i'za)
  • 兽一市：兽一市(se'i'shi)
  • 列车：车(sya)
  • 到达：至(ji)
  • 教程：文学(wen'se)
  • 完工：行毕(shin'bi)/竣(jun)
  • 赛所(sai'so)：赛场
  • 车公：公交车
  • 农所：农场
  • 钘：解锁

答案

  • 下一站者、兽一市也。车者、兽一市以至之时者、朝7时40分也。
  • 人历2022年8月11日时、此文学竣。
  • 每个季节中午十二点整，请检查赛场上的灌木丛。
  • 在冬季的早晨6点至下午4点之间，玩家从农场走到巴士站时解锁。
