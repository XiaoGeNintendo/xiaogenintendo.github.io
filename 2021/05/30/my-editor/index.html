My Software Preference

Editors, IDES, Workspaces are very important for a coder. And so is true for artists and composers.

In this article, I will share some of my editor/software preferences that I am using.

C/C++

  • Dev-C++

I didn’t install VC distribution for VS because it’s too big. I seldom do C++ developing either.

Java

  • Eclipse
  • Idea

I use Eclipse more often

HTML/JS/CSS

  • VSC
  • SublimeText

I use Sublime Text at school, Visual Studios Code at home.

Python

  • VSC
  • Idle

I use Idle at school

Danmakufu(DNH)

  • VSC

I have NPP plugin installed but VSC is better

Lua

  • VSC

C#

  • VS

Visual Studio is actually very nice to use and nice-looking too! It’s just not friendly to small-disk users.

Node.js

  • Repl.it

I’ve never written Node.js offline

Kotlin

  • Idea

I think no one uses Eclipse for Kotlin

Ruby

  • VSC

I have forgotten all Ruby now

Music making

  • FL Studio
  • Garage Band
  • Famitracker

GB has better sampling as I am lazy to pay more for FL’s.

Audio Editing

  • Audition
  • Audacity

I don’t know how to use Audacity

Drawing

  • MsPaint
  • Krita
  • Photoshop

I don’t draw on computer.

Image Editing

  • Paint.net
  • MsPaint

Free Basic (Legacy)

  • FBIde
