2021 Typing Game Release

Happy Chinese New Year!!

This year I bring you the ultimate Typing Game: 2021 Typing Game!

This game is based on Touhou Typing Engine, the same engine that made Touhou Rune Legend. And I modified it to be suitable for this game. In this game, you will experience a STG-style TPG. Be amazed at how powerful simple patterns in STG are in TPG!!

The game comes in 3 languages. Chinese and Japanese are Baidu translated.

You need Java 8 to run this software
